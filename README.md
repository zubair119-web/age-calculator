# Age Calculator App

A simple mobile-friendly age calculator that works in any browser.

## Features
- Calculate exact age in years, months, days
- Zodiac sign display
- Next birthday countdown
- Mobile responsive design
- Share with friends

## How to Use
1. Select your birth date
2. Select target date (default is today)
3. Click Calculate Age
4. Share with friends using the share button
